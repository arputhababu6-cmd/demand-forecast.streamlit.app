# Demand Forecasting System

This project predicts future demand using machine learning.

## Features
- Upload CSV data
- Clean data
- Predict demand
- Show graphs

## Tech Stack
Python, Pandas, Scikit-learn, Streamlit

## How to run
1. Install requirements
2. Run: streamlit run app.py
